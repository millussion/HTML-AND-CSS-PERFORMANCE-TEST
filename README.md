# ✦ CAMILLE MARRUGO - HTML AND CSS PERFORMANCE TEST


This repository contains the source code for the HTML and CSS performance test. 
---
This project is a **starter template** for a performance assessment focused on building a responsive landing page for a rock band.

The goal is to transform this base into a **visually engaging, responsive, and well-structured website** that promotes the band, their identity, and upcoming events.

## Visual Concept

This page follows a modern and cutting-edge aesthetic with an emphasis on:
* **Typographic Design:** Eye-catching headings and experimental designs.
* **Interactive Elements:** Custom side navigation.
* **Movement:** Smooth text animations and transitions that reflect the band's creative energy.
---

## Key Sections

* **Hero / Header:** An abstract high-impact intro.
* **Tour Information:** A header displaying tour information.
* **Dates:** A custom table showing dates, venues, and cities.
* **About Us:** A section where the band introduces themselves and provides information about the band.
* **Gallery:** A gallery of their photos along with a video of them.

---

##  Built With

* **HTML5:** Structured with semantic elements for better accessibility.
* **CSS3:** Custom styling (via `index.css`) including marquee animations, grid layouts and responsive sidebars.
* **JavaScript:** Powering the navigation logic (show/hide sidebar).
